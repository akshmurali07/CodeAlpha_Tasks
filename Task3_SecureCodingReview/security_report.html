CRITICAL V001 SQL Injection

Description: User input is directly embedded into SQL queries using f-strings, allowing attackers to manipulate the query logic.

Vulnerable Code (Line 39):

query = f"SELECT * FROM users WHERE username = '{username}' AND password = '{password}'"

Attack Example: Entering ' OR '1'='1 as username bypasses authentication entirely.

Fixed Code:

query = "SELECT * FROM users WHERE username = ? AND password = ?" result = conn.execute(query, (username, password)).fetchone()
HIGH V002 Hardcoded Credentials

Description: Passwords and secret keys are hardcoded directly in the source code. Anyone with access to the code can steal these credentials.

Vulnerable Code (Lines 9–12):

SECRET_KEY = "admin123" DB_PASSWORD = "password123" ADMIN_PASS = "admin123"

Fixed Code:

import os SECRET_KEY = os.environ.get("SECRET_KEY") DB_PASSWORD = os.environ.get("DB_PASSWORD") ADMIN_PASS = os.environ.get("ADMIN_PASS")
HIGH V003 Weak Hashing Algorithm (MD5)

Description: MD5 is cryptographically broken and unsuitable for password hashing. Attackers can reverse MD5 hashes using rainbow tables.

Vulnerable Code (Line 16):

return hashlib.md5(password.encode()).hexdigest()

Fixed Code:

import bcrypt def hash_password(password): return bcrypt.hashpw(password.encode(), bcrypt.gensalt()) def verify_password(password, hashed): return bcrypt.checkpw(password.encode(), hashed)
HIGH V004 Cross-Site Scripting (XSS)

Description: User input from the URL is directly injected into HTML output without sanitization, allowing attackers to run malicious JavaScript.

Vulnerable Code (Line 52):

return f"<h2>Search results for: {query}</h2>"

Attack Example: Visiting /search?q=<script>alert('hacked')</script> executes JavaScript in the victim's browser.

Fixed Code:

from markupsafe import escape return f"<h2>Search results for: {escape(query)}</h2>"
CRITICAL V005 Path Traversal

Description: User-controlled filename is used directly in a file path, allowing attackers to read any file on the system.

Vulnerable Code (Line 58):

filepath = os.path.join("uploads", filename)

Attack Example: Requesting /file?name=../../etc/passwd reads the system password file.

Fixed Code:

safe_name = os.path.basename(filename) filepath = os.path.join("uploads", safe_name) if not os.path.exists(filepath): return "File not found", 404
MEDIUM V006 Debug Mode Enabled in Production

Description: Flask's debug mode is enabled, which exposes an interactive Python debugger to anyone who triggers an error — giving full server access.

Fixed Code:

import os app.run(debug=os.environ.get("FLASK_DEBUG", "False") == "True")
