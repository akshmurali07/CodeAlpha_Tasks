# CodeAlpha Cyber Security Internship Tasks

This repository contains all the projects completed as part of the **CodeAlpha Cyber Security Internship**. Three tasks were completed covering network analysis, security awareness, and secure code review.

---

## Task 1: Basic Network Sniffer

A real-time network packet analyzer built with Python and Flask. The tool captures live network traffic and displays source/destination IPs, protocol types, port information, and packet size on a live dashboard with charts.

**Tech Stack:** Python, Scapy, Flask, Flask-SocketIO, Chart.js

**Files:** `Task1_NetworkSniffer/`

---

## Task 2: Phishing Awareness Training

An interactive web-based training module to help users recognize and avoid phishing attacks. Features a clickable fake email with red flag explanations, social engineering tactics breakdown, best practices checklist, and a scored quiz.

**Tech Stack:** HTML, CSS, JavaScript

**Live Demo:** https://akshmurali07.github.io/CodeAlpha_PhishingAwareness/

**Files:** `Task2_PhishingAwareness/`

---

## Task 3: Secure Coding Review

A security audit of a vulnerable Python web application. Includes a static analysis scanner that identifies common vulnerabilities and generates a detailed HTML report with findings and remediation steps.

**Tech Stack:** Python, HTML

**Files:** `Task3_SecureCodingReview/`
