# Task 2: Phishing Awareness Training

An interactive web-based phishing awareness training module built as part of the **CodeAlpha Cyber Security Internship**.

## Overview

Instead of just listing tips, this module lets users actively interact with phishing examples to understand exactly what makes them dangerous. It was inspired by a real incident where a family member almost fell victim to a phone scam after losing their credit card.

## Features

- **What is Phishing?** — Clear introduction to phishing and how it works
- **Spot the Phish** — A realistic fake email where clicking on suspicious parts reveals why they are red flags
- **Social Engineering Tactics** — Cards explaining urgency, authority, fear, and curiosity tactics
- **Best Practices Checklist** — Practical tips to stay safe online
- **Interactive Quiz** — 5 scenario-based questions with instant scoring and answer feedback

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Live Demo

https://akshmurali07.github.io/CodeAlpha_PhishingAwareness/

## How to Run

No setup needed. Simply open `index.html` in any web browser.
